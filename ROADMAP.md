# Spektra Product Manager Roadmap

Ciel: vlastny PIM system pre Spektra Install, ktory naplni a aktualizuje e-shop produktmi, cenami, popismi, SEO a obrazkami.

## Faza 1 - Core PIM

- Univerzalny produktovy model
- SQLite databaza produktov
- Import PDF/Excel/CSV/XML
- Zakladna detekcia dodavatela
- Export Shoptet CSV
- Zakladna Windows aplikacia

## Faza 2 - Produktovy editor

- Tabulka produktov v aplikacii
- Vyhladavanie a filtrovanie
- Editacia nazvu, ceny, kategorie, popisu
- Hromadne upravy
- Kontrola chyb pred exportom

## Faza 3 - AI a obrazky

- Generovanie kratkych popisov
- Generovanie dlhych HTML popisov
- SEO title a meta description
- Navrh kategorii a parametrov
- Automaticke hladanie a ukladanie obrazkov

## Faza 4 - Synchronizacia

- Shoptet API
- Aktualizacia cien a skladov
- Detekcia novych, zmenenych a vyradenych produktov
- Historia cien
- Export len zmenenych produktov

## Priorita

1. Stabilny import tabulkovych cennikov do databazy.
2. Stabilny export databazy do Shoptetu.
3. Editor produktov.
4. AI popisy a obrazky.
5. Priame Shoptet API.
